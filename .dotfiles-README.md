# Dotfiles (bare repo)

Managed with a bare git repo — files tracked in-place at their native `$HOME` paths.

## New Machine Setup (one command)

```bash
curl -fsSL https://raw.githubusercontent.com/NetanelK/dotfiles/main/.dotfiles-bootstrap.sh | bash
```

This installs Homebrew, clones the bare repo, checks out all configs, installs all
brew packages, oh-my-zsh + plugins + theme, and VS Code extensions.

## Daily Usage

```bash
dotfiles status                    # what changed
dotfiles add ~/.some-config        # track a file
dotfiles commit -m "update X"     # commit
dotfiles push                      # push to GitHub
```

The `dotfiles` alias is defined in `~/.env.zsh`:
```bash
alias dotfiles='git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
```

## Manual Setup on New Machine (alternative)

```bash
git clone --bare https://github.com/NetanelK/dotfiles.git $HOME/.dotfiles
alias dotfiles='git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
dotfiles config --local status.showUntrackedFiles no
dotfiles config --local core.excludesFile $HOME/.dotfiles-gitignore
dotfiles checkout
```

## What's Tracked

| Path | Content |
|------|---------|
| `.zshrc` | Oh-my-zsh, spaceship theme, plugins |
| `.env.zsh` | Env vars, aliases, NVM, Go path, fzf |
| `.gitconfig` | Identity, aliases, pull rebase |
| `.ssh/config` | SSH hosts (no private keys) |
| `.aws/config` | AWS profiles via saml2aws |
| `.saml2aws` | Okta federation config |
| `.claude/` | Hooks, rules, statusline, settings template |
| `.Brewfile` | All Homebrew formulae & casks |
| `.vscode-extensions.txt` | VS Code extension IDs |
| `Library/.../Code/User/settings.json` | VS Code settings |

## Secrets (NOT tracked)

- `~/.ssh/id_ed25519`, `~/.ssh/*.pem` — SSH keys
- `~/.claude/settings.json` — contains auth token (use `.template`)
- `~/.claude/zscaler-root-ca.pem` — corporate CA cert
- `~/.aws/credentials` — AWS session tokens

## Updating Brewfile

```bash
brew bundle dump --global --force
dotfiles add ~/.Brewfile && dotfiles commit -m "update Brewfile" && dotfiles push
```

## Updating VS Code Extensions

```bash
code --list-extensions | sort > ~/.vscode-extensions.txt
dotfiles add ~/.vscode-extensions.txt && dotfiles commit -m "update vscode exts" && dotfiles push
```
