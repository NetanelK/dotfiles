# Dotfiles (bare repo)

Portable, org-agnostic dev environment. Org-specific config goes in `.local` files (gitignored).

## New Machine Setup

```bash
curl -fsSL https://raw.githubusercontent.com/NetanelK/dotfiles/main/.dotfiles-bootstrap.sh | bash
```

## What's Tracked (generic)

| File | Content |
|------|---------|
| `.zshrc` | Oh-my-zsh, spaceship, plugins (sources `.zshrc.local`) |
| `.env.zsh` | Env vars, aliases, NVM, Go, fzf (sources `.env.local`) |
| `.gitconfig` | Identity, pull rebase, aliases (includes `.gitconfig.local`) |
| `.ssh/config` | Generic SSH patterns (includes `config.local`) |
| `.claude/` | Hooks, rules, statusline, settings template |
| `.Brewfile` | Homebrew packages |
| `.vscode-extensions.txt` | VS Code extensions |
| `.macos-defaults.sh` | macOS system preferences |
| `Library/.../Code/User/settings.json` | VS Code settings |

## What's NOT Tracked (org-specific `.local` files)

| File | Content |
|------|---------|
| `~/.zshrc.local` | Org exports (OTEL, certs, API tokens) |
| `~/.env.local` | Org aliases (awsconsole, etc.) |
| `~/.gitconfig.local` | Org email, autolinks |
| `~/.ssh/config.local` | Org SSH hosts, jump servers |
| `~/.aws/config` | AWS profiles |
| `~/.saml2aws` | SAML federation config |
| `~/.claude/settings.json` | Auth token, org plugins |

## How the `.local` Pattern Works

Each tracked config sources/includes its `.local` counterpart:

```
.zshrc         →  [ -f ~/.zshrc.local ] && source ~/.zshrc.local
.env.zsh       →  [ -f ~/.env.local ] && source ~/.env.local
.gitconfig     →  [include] path = ~/.gitconfig.local
.ssh/config    →  Include ~/.ssh/config.local
```

When you change orgs: create new `.local` files, keep everything else.

## Daily Usage

```bash
dotfiles status
dotfiles add ~/.some-config
dotfiles commit -m "update X"
dotfiles push
```

## Updating Brewfile / Extensions

```bash
brew bundle dump --global --force
dotfiles add ~/.Brewfile && dotfiles commit -m "update Brewfile" && dotfiles push

code --list-extensions | sort > ~/.vscode-extensions.txt
dotfiles add ~/.vscode-extensions.txt && dotfiles commit -m "update exts" && dotfiles push
```
